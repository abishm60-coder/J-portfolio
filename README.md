# J Portfolio

A premium, interactive web portfolio designed with a physical-meets-digital tactile aesthetic. Built using React + TypeScript, Vite, and pure CSS.

## 🎨 Visual & Layout Specifications

- **Pure Black Canvas (`#000000`)**: Deep, high-contrast black backdrop accented with a warm studio grey (`#121212`) folder container layout.
- **Signature Orange Accents (`#ff5500`)**: Eye-catching highlights for key typography, interactive indicators, active states, and custom handwriting annotations.
- **Perfect Typographic Alignment**: Paired elegant **Bebas Neue** headings, clean and functional **Inter** for UI copy, **JetBrains Mono** for status flags, and custom handwriting strokes via **Caveat**.
- **Tactile Elements**: Real-time rotating polaroid cards, physical sprocket hole filmstrips, interactive 3D folders, custom handwritten arrows, and torn paper vector shapes.

## 📂 Codebase Structure

- `/src/components/Header.tsx`: Fixed transparent blur menu.
- `/src/components/Hero.tsx`: Asymmetric polaroids with top relative handwriting links.
- `/src/components/SelectedCreations.tsx`: Polaroid cards, torn paper effects, and floating circular brand badge.
- `/src/components/About.tsx`: Horizontal filmstrip gallery with circular story trigger.
- `/src/components/Expertise.tsx`: Interactive physical folder layout mimicking standard workspace folders.
- `/src/components/CreativePlayground.tsx`: Clean masonry grid with unique handwritten pointers.
- `/src/components/ContactFooter.tsx`: Bold social layout, email channels, and pulse green active indicators.
- `/src/index.css`: Core layout styling rules.
- `/README.md`: Project documentation.
